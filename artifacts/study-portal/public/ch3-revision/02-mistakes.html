Mistake 1 Most Common

Order गलत लिखना — Columns पहले, Rows बाद

Pressure में बच्चे Rows और Columns उलट देते हैं। यह सबसे basic लेकिन marks काटने वाली गलती है।

❌ गलत

$\begin{bmatrix}1&2&3\\4&5&6\end{bmatrix}$ का order = 3×2 (columns पहले लिख दिया)

✅ सही

Rows = 2 (ऊपर से नीचे), Columns = 3 (बाएं से दाएं)
Order = 2×3 — पहले Rows, फिर Columns।

🧠 RC Cola Trick — कभी नहीं भूलोगे Row पहले, Column बाद — RC Cola! जैसे Cola की bottle में RC पहले आता है।
Mistake 2 Most Common

AB = BA मान लेना

सबसे बड़ी गलती। Numbers में $3\times4 = 4\times3$ होता है — बच्चे यही matrices में भी लगा देते हैं। Matrices में यह बिल्कुल गलत है।

❌ गलत सोच

अगर $AB$ निकाल लिया तो $BA$ automatically वही होगा।

✅ Proof — खुद देख लो
$$A=\begin{bmatrix}1&2\\3&4\end{bmatrix},\ B=\begin{bmatrix}0&1\\1&0\end{bmatrix}$$

$AB=\begin{bmatrix}2&1\\4&3\end{bmatrix}$    लेकिन    $BA=\begin{bmatrix}3&4\\1&2\end{bmatrix}$
AB ≠ BA — दोनों बिल्कुल अलग।

📌 Rule Matrix multiplication commutative नहीं है। Exam में अगर prove करने बोलें कि $AB \neq BA$, तो दोनों निकालो और दिखाओ।
Mistake 3 Exam Important

Multiplication से पहले Compatibility Check नहीं करना

बिना सोचे AB निकालने लग जाते हैं — लेकिन पहले check करना ज़रूरी है।

❌ गलत

$A$ (2×2) और $B$ (1×3) को directly multiply करना शुरू कर देना।

✅ सही तरीका

Check करो: A के columns = B के rows?
A(2×2) और B(1×3) → 2 ≠ 1 → AB defined नहीं।
A(2×3) और B(3×4) → 3 = 3 → AB possible। Result = 2×4

⚡ Easy Trick $A(m\times\underline{n}) \times B(\underline{n}\times p)$ — बीच के दोनों numbers same होने चाहिए। Result = $m\times p$।
Mistake 4 Exam Important

$(AB)' = A'B'$ लिखना — Order Reverse नहीं करना

Transpose "distribute" होता है — students यह सोचकर $A'B'$ लिख देते हैं। यह 2-4 marks काटने वाली गलती है।

❌ गलत
$$(AB)' = A'B'$$
✅ सही Formula
$$(AB)' = B'A' \quad \longleftarrow \text{Order REVERSE हो जाती है!}$$

पहले $B$ का transpose, फिर $A$ का।

🧠 जूते-मोज़े Trick जूते पहनते हो → पहले मोज़े। उतारते हो → पहले जूते (उल्टा क्रम)।
AB में पहले A, फिर B था → Transpose में पहले B', फिर A'।
Extra: $(ABC)' = C'B'A'$ — तीन हों तो भी सब उल्टा।
Mistake 5

Transpose में नया Order नहीं लिखना

Elements सही transpose करते हैं, लेकिन नया order लिखना भूल जाते हैं — या गलत order लिखते हैं।

❌ गलत
$$A=\begin{bmatrix}1&2&3\\4&5&6\end{bmatrix}_{2\times3} \implies A'=\begin{bmatrix}1&4\\2&5\\3&6\end{bmatrix}_{\mathbf{2\times3}}$$

Elements सही हैं — लेकिन order 2×3 ही रहने दिया।

✅ सही
$$A'=\begin{bmatrix}1&4\\2&5\\3&6\end{bmatrix}_{\mathbf{3\times2}}$$

Order पलट जाता है: $m\times n \rightarrow n\times m$। यहाँ 2×3 → 3×2।

📌 Rule Transpose लेने के बाद order लिखना ज़रूरी है — $m\times n$ matrix का transpose $n\times m$ होता है।
Mistake 6

$(AB)^{-1} = A^{-1}B^{-1}$ लिखना

Inverse में भी Transpose जैसा ही rule है — order reverse होती है। Students यह भूल जाते हैं।

❌ गलत
$$(AB)^{-1} = A^{-1}B^{-1}$$
✅ सही
$$(AB)^{-1} = B^{-1}A^{-1} \quad \longleftarrow \text{Order REVERSE}$$
🧠 एक साथ याद करो Transpose: $(AB)' = B'A'$ ← reverse
Inverse: $(AB)^{-1} = B^{-1}A^{-1}$ ← reverse
दोनों में एक ही pattern — order उलट जाता है।
Mistake 7 Exam Important

Skew Symmetric में Diagonal 0 भूलना

Skew Symmetric matrix बनाते या check करते वक्त diagonal elements 0 होने की condition ignore हो जाती है।

❌ गलत Skew Symmetric
$$A=\begin{bmatrix}2&3\\-3&5\end{bmatrix}$$

Diagonal पर 2 और 5 हैं — यह skew symmetric नहीं है।

✅ सही Skew Symmetric
$$A=\begin{bmatrix}0&3\\-3&0\end{bmatrix}$$

Proof: यदि $A' = -A$ तो $a_{ii} = -a_{ii}$ → $2a_{ii}=0$ → $a_{ii}=0$
Diagonal हमेशा 0 — यह prove होता है।

⚡ Quick Check किसी matrix को देखते ही — पहले diagonal देखो। Diagonal में कोई भी element 0 नहीं है? तो skew symmetric हो ही नहीं सकता।
Mistake 8

अलग Order की Matrices जोड़ने की कोशिश

Addition के लिए order same होना ज़रूरी है — pressure में यह rule भूल जाते हैं।

❌ गलत
$$A=[1\ 2\ 3]_{1\times3},\quad B=[4\ 5]_{1\times2}$$

$A+B = [5\ 7\ 3]$ — यह बिल्कुल गलत है, possible ही नहीं।

✅ सही

A(1×3) और B(1×2) — same order नहीं।
सीधे लिखो: "A + B is not defined" — इससे ज़्यादा कुछ नहीं।

📌 Rule Addition और Subtraction सिर्फ same order की matrices में। Multiplication के लिए अलग rule है।
Mistake 9

Equality में Order Check किए बिना Elements Compare करना

दो matrices equal हैं या नहीं — पहले order देखो, elements बाद में।

❌ गलत approach
$$A=\begin{bmatrix}1&2&3\end{bmatrix}_{1\times3},\quad B=\begin{bmatrix}1&2\\3&4\end{bmatrix}_{2\times2}$$

"1=1, 2=2... elements same हैं तो equal होगा!" — गलत।

✅ सही Step-by-Step

Step 1: Order check — A(1×3), B(2×2) → different order → A ≠ B। Done।
Elements देखने की ज़रूरत ही नहीं। Order अलग है तो बराबर हो ही नहीं सकते।

📌 Equality के दो conditions 1. Same order   2. हर corresponding element बराबर ($a_{ij} = b_{ij}$ सब के लिए)
दोनों ज़रूरी — एक भी fail हो तो matrices equal नहीं।
Mistake 10

$kA$ में कुछ Elements को Multiply नहीं करना

Scalar multiply करते वक्त जल्दी में कुछ elements छूट जाते हैं।

❌ गलत
$$A=\begin{bmatrix}2&4\\6&8\end{bmatrix},\quad 2A=\begin{bmatrix}4&8\\6&8\end{bmatrix}$$

नीचे वाली row multiply नहीं की!

✅ सही
$$2A=\begin{bmatrix}2\times2&2\times4\\2\times6&2\times8\end{bmatrix}=\begin{bmatrix}4&8\\12&16\end{bmatrix}$$

हर एक element multiply होता है — एक भी नहीं छोड़ना।

Mistake 11 Bihar Board Trap!

$A^2$ को Elements का Square समझना

यह गलती 9वीं-10वीं की पुरानी habit से आती है और Bihar Board में हर साल यह सवाल आता है।

❌ गलत — बहुत बड़ी गलती
$$A=\begin{bmatrix}1&2\\3&4\end{bmatrix} \implies A^2=\begin{bmatrix}1^2&2^2\\3^2&4^2\end{bmatrix}=\begin{bmatrix}1&4\\9&16\end{bmatrix}$$

Elements को individually square कर दिया — यह scalar operation है, matrix नहीं!

✅ सही — A² = A × A (Matrix Multiplication)
$$A^2 = A\times A = \begin{bmatrix}1&2\\3&4\end{bmatrix}\times\begin{bmatrix}1&2\\3&4\end{bmatrix}$$
$$=\begin{bmatrix}1\times1+2\times3 & 1\times2+2\times4\\3\times1+4\times3 & 3\times2+4\times4\end{bmatrix}=\begin{bmatrix}7&10\\15&22\end{bmatrix}$$
🎯 Exam Alert Bihar Board में "$A^2$, $A^3$ निकालो" वाले सवाल हर साल आते हैं। हमेशा याद रखो — $A^n$ मतलब $A$ को खुद से $n$ बार matrix multiplication करना।
Mistake 12 Concept Trap

$AB = O$ का मतलब $A = O$ या $B = O$ समझना

Numbers में अगर $a \times b = 0$ है तो $a = 0$ या $b = 0$ — यही rule students matrices पर लगा देते हैं। Matrix में यह बिल्कुल गलत है।

❌ गलत सोच

$AB = O$ दिया है → "तो ज़रूर $A = O$ या $B = O$ होगा।"

✅ Counter Example — Proof
$$A=\begin{bmatrix}1&0\\0&0\end{bmatrix},\quad B=\begin{bmatrix}0&0\\1&1\end{bmatrix}$$
$$AB=\begin{bmatrix}1\times0+0\times1&1\times0+0\times1\\0\times0+0\times1&0\times0+0\times1\end{bmatrix}=\begin{bmatrix}0&0\\0&0\end{bmatrix}=O$$

लेकिन $A \neq O$ और $B \neq O$ — दोनों में non-zero elements हैं!

📌 Rule $AB = O$ → इसका मतलब ज़रूरी नहीं कि $A = O$ या $B = O$।
Matrix में zero product property नहीं चलती।
Mistake 13 Exam Important

Identity, Scalar, Diagonal Matrix में अंतर नहीं समझना

Bihar Board directly पूछता है — "यह कौन सा matrix है?" तीनों को clearly जानना ज़रूरी है।

❌ गलत

"$\begin{bmatrix}3&0\\0&3\end{bmatrix}$ एक Identity Matrix है।" — गलत।

✅ सही Hierarchy
Diagonal: बाकी सब 0, diagonal = कुछ भी  →  $\begin{bmatrix}2&0\\0&5\end{bmatrix}$
Scalar: Diagonal + सभी diagonal elements equal  →  $\begin{bmatrix}3&0\\0&3\end{bmatrix}$
Identity: Scalar + diagonal = 1  →  $\begin{bmatrix}1&0\\0&1\end{bmatrix}$
📌 Relation (याद करो) Identity ⊂ Scalar ⊂ Diagonal ⊂ Square
हर Identity → Scalar है। हर Scalar → Diagonal है।
उल्टा नहीं — हर Diagonal, Scalar नहीं। हर Scalar, Identity नहीं।
Mistake 14

$a_{ij}$ और $a_{ji}$ में Confusion — Element गलत जगह से लेना

Element notation उलटा समझने से transpose, symmetric — सब गलत हो जाता है।

❌ गलत

"$a_{23}$ = 3rd row, 2nd column का element।"

✅ सही
$$A=\begin{bmatrix}1&2&3\\4&5&\mathbf{6}\\7&8&9\end{bmatrix} \implies a_{23}=\mathbf{6}\ (\text{2nd row, 3rd column})$$

$a_{ij}$ → i = row, j = column। पहले row, फिर column।
Transpose में: $A'=[a_{ji}]$ → row और column उलट जाते हैं।

Mistake 15

Inverse सिर्फ Square Matrix का होता है — यह भूलना

Students किसी भी matrix का inverse निकालने की कोशिश करने लगते हैं।

❌ गलत

$A=\begin{bmatrix}1&2&3\\4&5&6\end{bmatrix}$ (2×3 matrix) का inverse निकालने की कोशिश।

✅ सही

Inverse सिर्फ Square Matrix (m = n) का हो सकता है।
2×3 matrix का inverse कभी नहीं होगा।
Square matrix का भी inverse तभी होगा जब row operations में कोई row all-zero न बने।

⚠️ Inverse exist नहीं करता जब Elementary row operations करते वक्त किसी भी row के सभी elements 0 हो जाएँ → वहीं रुको, लिखो: "Inverse does not exist।"
