Mistake 1 — सबसे Common

AB = BA समझ लेना

यह chapter की सबसे बड़ी गलती है। बहुत से students numbers की तरह सोचते हैं — "3×5 = 5×3, तो AB भी BA के बराबर होगा।" लेकिन matrices में यह बिल्कुल गलत है।

❌ गलत सोच

अगर $AB = \begin{bmatrix}1&2\\3&4\end{bmatrix}$ है, तो $BA$ भी यही होगा।

✅ सही बात
$$A=\begin{bmatrix}1&2\\3&4\end{bmatrix},\ B=\begin{bmatrix}0&1\\1&0\end{bmatrix}$$

$AB = \begin{bmatrix}2&1\\4&3\end{bmatrix}$    लेकिन    $BA = \begin{bmatrix}3&4\\1&2\end{bmatrix}$
AB ≠ BA — दोनों बिल्कुल अलग हैं।

📌 याद रखो Matrix multiplication generally commutative नहीं होती। सवाल में अगर AB दिया है तो BA automatically नहीं लिख सकते।
Mistake 2

Multiplication से पहले Compatibility Check नहीं करना

बहुत से students directly multiply करना शुरू कर देते हैं बिना यह check किये कि multiplication possible है या नहीं।

❌ गलत

$A$ (2×3) और $B$ (2×4) को directly multiply करने लगना।
ऐसे: $(AB)_{11} = a_{11}b_{11} + \ldots$ — यह सब गलत है।

✅ सही तरीका

पहले check करो: A के columns = B के rows?
A(2×3) और B(2×4) → 3 ≠ 2 → AB possible नहीं।

A(2×3) और B(3×4) → 3 = 3 → AB possible। Result = 2×4 matrix।

⚡ RC Trick A(m×n) × B(n×p) — बीच वाले दोनों n बराबर होने चाहिए। Result = m×p।
Mistake 3

Order लिखने में Columns पहले, Rows बाद

Order याद करते वक्त अक्सर उलझन होती है — पहले row लिखें या column?

❌ गलत

$\begin{bmatrix}1&2&3\\4&5&6\end{bmatrix}$ का order = 3×2 (पहले columns लिख दिया)

✅ सही

इस matrix में 2 rows और 3 columns हैं।
Order = 2×3 — पहले Rows (2), फिर Columns (3)।

🧠 RC Cola Trick Row पहले, Column बाद — RC Cola! कभी नहीं भूलोगे।
Mistake 4 — Exam में बहुत आता है

$(AB)' = A'B'$ लिखना — Order Reverse नहीं करना

Transpose लेते वक्त students सीधे $A'B'$ लिख देते हैं — यह गलती exam में marks काटती है।

❌ गलत

$(AB)' = A'B'$ — order same रखना।

✅ सही

$(AB)' = \mathbf{B'A'}$ — order reverse होती है।

$$\text{जैसे जूते पहनते हैं तो मोज़े पहले — उतारते हैं तो जूते पहले}$$

पहले B का transpose, फिर A का। यही order है।

📌 एक साथ याद करो $(AB)' = B'A'$ और $(ABC)' = C'B'A'$ — सब उल्टा।
$(A+B)' = A'+B'$ — यहाँ order नहीं बदलती।
Mistake 5

$(AB)^{-1} = A^{-1}B^{-1}$ लिखना

Transpose की तरह ही Inverse में भी order reverse होती है — students यह भूल जाते हैं।

❌ गलत

$(AB)^{-1} = A^{-1}B^{-1}$

✅ सही

$(AB)^{-1} = \mathbf{B^{-1}A^{-1}}$ — order reverse होती है।

🧠 एक साथ याद करो Transpose में: $(AB)' = B'A'$ ← reverse
Inverse में: $(AB)^{-1} = B^{-1}A^{-1}$ ← reverse
दोनों में एक ही rule — order उलट जाता है।
Mistake 6

$a_{ij}$ और $a_{ji}$ में Confusion

Element का notation गलत समझने से पूरा सवाल गलत हो जाता है।

❌ गलत

$a_{23}$ = 3rd row, 2nd column का element।

✅ सही

$a_{23}$ = 2nd row, 3rd column का element।
$a_{ij}$ में — पहला subscript (i) = row, दूसरा (j) = column।

$$A=\begin{bmatrix}1&2&3\\4&5&\mathbf{6}\\7&8&9\end{bmatrix} \implies a_{23} = \mathbf{6}$$
📌 Transpose में $A' = [a_{ji}]$ — यहाँ i और j उलट जाते हैं।
मतलब A का $a_{23}$ = A' का $a_{32}$।
Mistake 7

Skew Symmetric में Diagonal 0 भूलना

Skew symmetric matrix का एक golden rule है जो students अक्सर prove करते वक्त ignore कर देते हैं।

❌ गलत — ऐसा matrix skew symmetric नहीं हो सकता
$$B=\begin{bmatrix}\mathbf{3}&2\\-2&\mathbf{5}\end{bmatrix}$$

यहाँ diagonal elements 3 और 5 हैं — यह skew symmetric नहीं है।

✅ सही Skew Symmetric Matrix
$$B=\begin{bmatrix}\mathbf{0}&3\\-3&\mathbf{0}\end{bmatrix}$$

Diagonal हमेशा 0 होता है। Proof: $a_{ii} = -a_{ii} \Rightarrow 2a_{ii}=0 \Rightarrow a_{ii}=0$

⚡ Quick Check किसी matrix को check करना हो कि skew symmetric है या नहीं — पहले diagonal देखो। Diagonal में 0 नहीं है तो skew symmetric हो ही नहीं सकता।
Mistake 8 — Definition Confusion

Identity, Scalar, Diagonal में अंतर न समझना

Bihar Board में directly पूछते हैं — "यह कौन सा matrix है?" Students इन तीनों को mix कर देते हैं।

❌ गलत सोच

"$\begin{bmatrix}3&0\\0&3\end{bmatrix}$ एक Identity matrix है।"

✅ सही अंतर
Diagonal Matrix: बाकी सब 0, diagonal कुछ भी → $\begin{bmatrix}2&0\\0&5\end{bmatrix}$
Scalar Matrix: Diagonal matrix + सभी diagonal elements equal → $\begin{bmatrix}3&0\\0&3\end{bmatrix}$
Identity Matrix: Scalar matrix + diagonal elements = 1 → $\begin{bmatrix}1&0\\0&1\end{bmatrix}$

Relation: Identity ⊂ Scalar ⊂ Diagonal ⊂ Square
📌 याद रखो हर Identity matrix, Scalar matrix है।
हर Scalar matrix, Diagonal matrix है।
लेकिन हर Diagonal matrix, Scalar नहीं। हर Scalar, Identity नहीं।
Mistake 9

अलग Order की Matrices जोड़ने की कोशिश

Addition के लिए order same होना ज़रूरी है — यह basic rule है जो pressure में भूल जाते हैं।

❌ गलत
$$A=\begin{bmatrix}1&2\\3&4\end{bmatrix}_{2\times2},\ B=\begin{bmatrix}1&2&3\\4&5&6\end{bmatrix}_{2\times3}$$

A+B का attempt करना — यह defined नहीं है।

✅ सही

A(2×2) और B(2×3) — different order → Addition possible नहीं।
Addition और Subtraction सिर्फ same order की matrices के बीच होती है।

Mistake 10

Symmetric Check गलत तरीके से करना

Students कभी-कभी बिना $A'$ निकाले ही कह देते हैं — "यह symmetric है।"

❌ गलत approach
$$A=\begin{bmatrix}1&2\\3&1\end{bmatrix}$$

"Diagonal same है, तो symmetric होगा।" — यह reasoning गलत है।

✅ सही तरीका — A' निकालो और compare करो
$$A'=\begin{bmatrix}1&3\\2&1\end{bmatrix} \neq A$$

$A' \neq A$ → यह symmetric नहीं है।
Symmetric के लिए ज़रूरी: $a_{ij} = a_{ji}$ सब के लिए — यानी $a_{12}=a_{21}$, $a_{13}=a_{31}$ etc.

📌 Correct Example $A=\begin{bmatrix}1&2\\2&5\end{bmatrix}$ → $A'=\begin{bmatrix}1&2\\2&5\end{bmatrix}=A$ → ✅ Symmetric।
यहाँ $a_{12}=a_{21}=2$ — यही condition है।
Mistake 11 — Inverse निकालते वक्त

Row और Column Operations एक साथ Mix करना

Inverse निकालते वक्त एक ही method use करना है — या तो सिर्फ Row operations, या सिर्फ Column operations। दोनों को mix करने से गलत answer आता है।

❌ गलत

$A = IA$ लिखकर कुछ Row operations करो, फिर बीच में Column operation भी कर दो।
यह बिल्कुल गलत है।

✅ सही

Row method: $A = IA$ लिखो → सिर्फ Row operations करो → जब A की जगह I आ जाए → दाईं तरफ $A^{-1}$

Column method: $A = AI$ लिखो → सिर्फ Column operations करो → जब A की जगह I आ जाए → दाईं तरफ $A^{-1}$

⚠️ Inverse exist नहीं करता जब Operations करते वक्त अगर किसी row (या column) के सभी elements 0 हो जाएँ → $A^{-1}$ exist नहीं करता। वहीं रुक जाओ और लिखो "Inverse does not exist।"
Mistake 12

Zero Matrix को "0" लिखना — "O" नहीं

यह छोटी लेकिन marks काटने वाली गलती है।

❌ गलत notation

$A + (-A) = \mathbf{0}$ या $AB = \mathbf{0}$

✅ सही notation

$A + (-A) = \mathbf{O}$ — यहाँ O (capital letter) = Zero Matrix
$AB = \mathbf{O}$ — Zero Matrix को O से लिखते हैं, अंक 0 से नहीं।

📌 और एक बात $AB = O$ का मतलब यह नहीं कि $A = O$ या $B = O$ — matrices में यह नियम नहीं चलता। यह numbers जैसा नहीं है।

  • M9Addition सिर्फ same order matrices में होती है
