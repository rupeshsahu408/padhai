Block 1

Matrix — Basic

  • Rows और Columns में सजी संख्याओं को Matrix कहते हैं — Square Brackets [ ] में
  • जैसे Excel sheet — rows नीचे जाती हैं, columns साइड में
  • Order = m × n → m = Rows, n = Columns | Total elements = m × n
  • $a_{ij}$ = i वीं Row, j वाँ Column का element  |  $a_{23}$ = 2nd row, 3rd column
  • 8 elements हों → possible orders: 1×8, 8×1, 2×4, 4×2
🧠 स्मरण सूत्र — RC Cola Order लिखते हो → पहले Row, फिर Column। RC Cola की तरह — R पहले आता है। 🥤
🧠 स्मरण सूत्र — $a_{ij}$ पहले मंजिल (row = i), फिर कमरा नंबर (column = j)। 🏠
Hotel में पहले floor देखते हैं, फिर room number।

Block 3

$I_2$ और $I_3$ — Direct याद करो

$I_2$ (2×2)
$$I_2 = \begin{bmatrix}1&0\\0&1\end{bmatrix}$$
$I_3$ (3×3)
$$I_3 = \begin{bmatrix}1&0&0\\0&1&0\\0&0&1\end{bmatrix}$$
  • $AI = IA = A$ — Identity से गुणा करने पर matrix नहीं बदलती
  • $AO = OA = O$ — Zero matrix से गुणा करने पर Zero ही मिलेगा
Block 4

समानता (Equality)

  • A = B तभी जब: ① Same Order और ② हर $a_{ij} = b_{ij}$
  • Order अलग है → तुरंत "Not Equal" — elements देखने की ज़रूरत नहीं
🧠 स्मरण सूत्र दो लोगों को compare करते हो → पहले कद मिलाओ (order), फिर चेहरा मिलाओ (elements)। 👥
Block 5

जोड़, घटाव, Scalar Multiplication

  • Addition/Subtraction: Same order ज़रूरी → $(A\pm B)_{ij} = a_{ij} \pm b_{ij}$
  • Different order → "Not Defined" — बस इतना लिखो
  • Scalar $kA$: हर एक element को k से गुणा — एक भी नहीं छोड़ना → $(kA)_{ij} = k\cdot a_{ij}$
  • $-A = (-1)A$  |  $A - B = A + (-1)B$
  • $A+B = B+A$ ✓ (Commutative)  |  $(A+B)+C = A+(B+C)$ ✓ (Associative)
  • $A + O = O + A = A$  |  $A + (-A) = O$
  • $k(A+B) = kA+kB$  |  $(k+l)A = kA+lA$
🧠 स्मरण सूत्र — Addition एक ही size के डिब्बे मिलाए जा सकते हैं — अलग size वाले नहीं। 📦
🧠 स्मरण सूत्र — Scalar kA Tax की तरह — हर चीज़ पर लगता है, कोई element बचता नहीं! 😄
Block 6

Matrix Multiplication — 3 Steps में

1
Check करो: $A(m\times\underline{n})\times B(\underline{n}\times p)$ → बीच के दोनों numbers same होने चाहिए। नहीं तो → "Not Defined"
2
Result का order: AB = $m\times p$ (बाहर के दो numbers)
3
Calculate: $(AB)_{ij}$ = A की i वीं row × B का j वाँ column → element-wise multiply करके जोड़ो
  • AB ≠ BA — Commutative नहीं! ✗
  • $(AB)C = A(BC)$ ✓ Associative  |  $A(B+C)=AB+AC$ ✓ Distributive
  • $AI = IA = A$ ✓  |  $AO = OA = O$ ✓
  • $AB = O$ का मतलब $A=O$ या $B=O$ नहीं! ✗
  • $A^2 = A\times A$ — Elements को square नहीं करते! ✗
🧠 स्मरण सूत्र — Compatibility बीच में हाथ मिलाना ज़रूरी है — $n = n$ match नहीं? तो multiplication नहीं होगी! 🤝
🧠 स्मरण सूत्र — AB ≠ BA Delhi से Patna का रास्ता, Patna से Delhi का रास्ता अलग हो सकता है।
Matrix में order matter करता है — जैसे पहले जूता पहनो, फिर मोज़ा नहीं चलता! 👟
Block 7

Transpose — $A'$ या $A^T$

  • Rows → Columns, Columns → Rows। Order पलटता है: $m\times n \to n\times m$
  • $A'=[a_{ji}]$ → Transpose में matrix को 45° तिरछा घुमा दो
PropertyFormula
Double Transpose$(A')' = A$
Scalar$(kA)' = kA'$
Addition$(A+B)' = A'+B'$
Multiplication$(AB)' = B'A'$← Order REVERSE ⚠️
Three matrices$(ABC)' = C'B'A'$← सब उल्टा
🧠 स्मरण सूत्र — जूते-मोज़े जूते उतारते हो → पहले जूता, फिर मोज़ा (उल्टा क्रम)।
$(AB)' = B'A'$ — पहले $B'$, फिर $A'$। उल्टा। 👟
Block 8

Symmetric और Skew Symmetric

Property Symmetric (सममित) Skew Symmetric (विषम सममित)
Condition$A' = A$$A' = -A$
Elements$a_{ij} = a_{ji}$$a_{ij} = -a_{ji}$
Diagonalकोई भी valueहमेशा 0
Example $\begin{bmatrix}1&2\\2&5\end{bmatrix}$ $\begin{bmatrix}0&3\\-3&0\end{bmatrix}$
  • $(A+A')$ → हमेशा Symmetric
  • $(A-A')$ → हमेशा Skew Symmetric
  • Most Important:  $A = \dfrac{1}{2}(A+A') + \dfrac{1}{2}(A-A')$ → Sym + Skew
🧠 स्मरण सूत्र — Symmetric Symmetric = आईना — बाईं तरफ जो है, दाईं तरफ भी वही। 🪞
Skew = उल्टा आईना — और diagonal पर खाली मैदान (= 0)! ⚽
🧠 स्मरण सूत्र — Sum Formula हर इंसान में अच्छाई + बुराई दोनों होती है।
हर Matrix = $\frac{1}{2}$(Symmetric) + $\frac{1}{2}$(Skew Symmetric)। 😄
Block 9

Elementary Operations — 3 Types

  • ① Swap: दो rows/columns की अदला-बदली → $R_i \leftrightarrow R_j$
  • ② Scale: Row/Column को non-zero k से गुणा → $R_i \to kR_i$, ($k\neq0$)
  • ③ Add: एक row में दूसरे का k गुना जोड़ो → $R_i \to R_i + kR_j$
🧠 स्मरण सूत्र — SSA Swap → Scale → Add। बस तीन काम। SSA याद करो।
Block 10

Invertible Matrix — Inverse

  • $AB = BA = I$ → $B = A^{-1}$ (A का inverse)
  • Inverse सिर्फ Square Matrix का  |  Inverse हमेशा Unique
  • $(AB)^{-1} = B^{-1}A^{-1}$ ← Order REVERSE!
  • $(A')^{-1} = (A^{-1})'$
  • Row Method: $A = IA$ → Row operations → Left side = I → Right side = $A^{-1}$
  • Column Method: $A = AI$ → Column operations → Left side = I → Right side = $A^{-1}$
  • Row और Column operations mix मत करो — एक ही method!
⚠️ Operations करते वक्त कोई row/column पूरी 0 हो जाए → लिखो: "Inverse does not exist।"
🧠 स्मरण सूत्र $(AB)^{-1} = B^{-1}A^{-1}$ — जूते उतारने का उल्टा क्रम।
AB में B बाद में था → Inverse में $B^{-1}$ पहले। 👟
Block 11

"Not Defined" — कब लिखते हैं?

  • Addition/Subtraction: दोनों matrices का order अलग हो
  • Multiplication AB: A के columns ≠ B के rows
  • Inverse: Matrix square नहीं है, या operations में row all-zero हो जाए
🧠 स्मरण सूत्र जब fit नहीं होता — "Not Defined" लिखो और आगे बढ़ो। ✌️
Half-answer लिखने से अच्छा है clearly लिखना: "Not Defined।"
