Block 1

Matrix — Basic

  • Rows और Columns में सजी संख्याओं को Matrix कहते हैं — Square Brackets [ ] में
  • Order = m × n → m = Rows, n = Columns
  • Total elements = m × n
  • $a_{ij}$ = i वीं Row, j वाँ Column का element
  • अगर 8 elements हों → possible orders: 1×8, 8×1, 2×4, 4×2
🧠 स्मरण सूत्र — RC Cola Order लिखते हो → पहले Row, फिर Column। RC Cola की तरह — R पहले आता है।
Block 2

Matrix के प्रकार — सब एक नज़र में

Type Condition Example
Row Matrix सिर्फ 1 Row (order 1×n) $[3\ -1\ 7]$
Column Matrix सिर्फ 1 Column (order m×1) $\begin{bmatrix}2\\5\end{bmatrix}$
Rectangular Rows ≠ Columns (m ≠ n) 2×3 matrix
Square Rows = Columns (m = n) 2×2, 3×3

जैसे: दादा → बाबा → पिता → बेटा। हर बड़ा छोटे को cover करता है।
Block 3

समानता (Equality)

  • A = B तभी जब: 1. Same Order हो और 2. हर $a_{ij} = b_{ij}$ हो
  • Order अलग है → बस लिखो "Not Equal" — elements देखने की ज़रूरत नहीं
🧠 स्मरण सूत्र जैसे दो जोड़ी जुते compare करते हो — पहले size check (order), फिर design check (elements)।
Block 4

जोड़, घटाव, Scalar Multiplication

  • Addition/Subtraction: Same order ज़रूरी → Corresponding elements को जोड़ो/घटाओ
  • $(A+B)_{ij} = a_{ij}+b_{ij}$  |  $(A-B)_{ij} = a_{ij}-b_{ij}$
  • Scalar kA: हर एक element को k से गुणा करो — एक भी नहीं छोड़ना
  • $(kA)_{ij} = k\cdot a_{ij}$  |  $-A = (-1)A$
  • Commutative: $A+B = B+A$ ✓
  • Associative: $(A+B)+C = A+(B+C)$ ✓
  • Identity: $A+O = A$  |  Inverse: $A+(-A) = O$
  • $k(A+B) = kA+kB$  |  $(k+l)A = kA+lA$
⚠️ $A+B$ possible नहीं अगर order अलग हो। "Not defined" लिखो।
Block 5

Matrix Multiplication — सबसे Important

  • AB possible तभी: A के columns = B के rows
  • $A(m\times n) \times B(n\times p)$ → $AB$ का order = $m\times p$
  • $(AB)_{ij} = \sum_{k=1}^{n} a_{ik}\cdot b_{kj}$ → i वीं row × j वाँ column, फिर जोड़ो
  • $A^2 = A\times A$ — matrix multiplication। Elements को square नहीं करते!
  • Associative: $(AB)C = A(BC)$ ✓
  • Distributive: $A(B+C) = AB+AC$ ✓
  • Identity: $AI = IA = A$
  • Commutative नहीं: $AB \neq BA$ generally ✗
  • Zero Product नहीं: $AB=O$ का मतलब $A=O$ या $B=O$ नहीं!
🧠 स्मरण सूत्र $A(m\times\underline{n})\times B(\underline{n}\times p)$ — बीच के दो numbers match होने चाहिए।
जैसे ताला और चाबी — साइज match करेगी तभी खुलेगा।
🧠 AB ≠ BA — स्मरण Traffic rule की तरह — Delhi से Patna का रास्ता, Patna से Delhi का रास्ता अलग हो सकता है।
Block 6

Transpose (परिवर्त) — $A'$ या $A^T$

  • Rows → Columns बना दो, Columns → Rows। Order पलट जाता है: $m\times n \to n\times m$
  • $A'=[a_{ji}]$ → i और j की जगह बदल जाती है
PropertyFormula
Double Transpose$(A')' = A$
Scalar$(kA)' = kA'$
Addition$(A+B)' = A'+B'$
Multiplication$(AB)' = B'A'$← Order REVERSE!
Three matrices$(ABC)' = C'B'A'$← सब उल्टा
🧠 स्मरण सूत्र — जूते-मोज़े जूते उतारते हो → पहले जूते, फिर मोज़े (उल्टा)।
$(AB)' = B'A'$ — पहले $B'$, फिर $A'$। उल्टा क्रम।
Block 7

Symmetric और Skew Symmetric

Property Symmetric (सममित) Skew Symmetric (विषम सममित)
Condition $A' = A$ $A' = -A$
$a_{ij}$ $a_{ij} = a_{ji}$ $a_{ij} = -a_{ji}$
Diagonal कोई भी value हमेशा 0
Example $\begin{bmatrix}1&2\\2&5\end{bmatrix}$ $\begin{bmatrix}0&3\\-3&0\end{bmatrix}$
  • $(A+A')$ → हमेशा Symmetric
  • $(A-A')$ → हमेशा Skew Symmetric
  • Most Important: $A = \dfrac{1}{2}(A+A') + \dfrac{1}{2}(A-A')$ → हर square matrix = Sym + Skew
🧠 स्मरण सूत्र — Skew Diagonal Skew Symmetric देखते ही → पहले diagonal check करो।
Diagonal में 0 नहीं? → Skew Symmetric हो ही नहीं सकता। Done।
🧠 स्मरण सूत्र — Sum Formula हर आदमी = उसका अच्छा आधा + बुरा आधा।
हर Matrix = $\frac{1}{2}$(Symmetric part) + $\frac{1}{2}$(Skew part)।
Block 8

Elementary Operations (प्रारंभिक संक्रियाएँ)

  • Type 1: दो rows/columns की अदला-बदली → $R_i \leftrightarrow R_j$
  • Type 2: Row/Column को non-zero k से गुणा → $R_i \to kR_i$, ($k\neq0$)
  • Type 3: एक row में दूसरे का k गुना जोड़ो → $R_i \to R_i + kR_j$
🧠 स्मरण सूत्र 3 operations — Swap, Scale, Add। SSA याद करो।
Block 9

Invertible Matrix — Inverse

  • $AB = BA = I$ → $B = A^{-1}$ (A का inverse)
  • Inverse सिर्फ Square Matrix का होता है
  • Inverse हमेशा Unique होता है
  • $(AB)^{-1} = B^{-1}A^{-1}$ ← Order REVERSE
  • $(A')^{-1} = (A^{-1})'$ — दोनों order में होता है
  • Row Method: $A = IA$ लिखो → Row operations → जब left side = I → right side = $A^{-1}$
  • Column Method: $A = AI$ लिखो → Column operations → जब left side = I → right side = $A^{-1}$
  • Row और Column operations mix मत करो — एक method ही use करो
⚠️ अगर कोई row/column पूरी 0 हो जाए → Inverse exist नहीं करता। लिखो: "Inverse does not exist।"
🧠 स्मरण सूत्र $(AB)^{-1} = B^{-1}A^{-1}$ याद नहीं रहता?
जूते पहनने का reverse — पहले जूता, फिर मोज़ा उतारो।
AB में B बाद में था → Inverse में B⁻¹ पहले आता है।
