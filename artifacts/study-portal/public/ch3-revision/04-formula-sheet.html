Topic 1

कोटि और संकेत (Order & Notation)

Formula का नाम Formula एक-line meaning (Hindi) Note / Condition
कोटि (Order) $m \times n$ m = Rows की संख्या, n = Columns की संख्या। RC Cola — Row पहले, Column बाद। m और n positive integers होते हैं
कुल अवयव (Total Elements) $m \times n$ Order का गुणनफल = matrix में कुल elements की संख्या 8 elements → orders: 1×8, 8×1, 2×4, 4×2
अवयव संकेत (Element Notation) $a_{ij}$ i वीं row और j वें column का element। "मंजिल (i) → कमरा (j)" $1 \le i \le m,\ 1 \le j \le n$
व्यापक रूप (General Form) $A = [a_{ij}]_{m \times n}$ A एक m×n matrix है जिसका (i,j)वाँ element $a_{ij}$ है Capital letter से matrix, small से element
Topic 2

आव्यूह के प्रकार (Types of Matrices)

प्रकार Condition Meaning / Example Special Note
पंक्ति आव्यूह (Row) Order = 1×n सिर्फ 1 row होती है। उदा: $[3\ -1\ 7]$ —
स्तम्भ आव्यूह (Column) Order = m×1 सिर्फ 1 column होता है। उदा: $\begin{bmatrix}2\\5\end{bmatrix}$ —
आयताकार आव्यूह (Rectangular) $m \neq n$ Rows और Columns की संख्या अलग-अलग —
वर्ग आव्यूह (Square) $m = n$ Rows = Columns। Inverse, Symmetric, Skew — सब Square matrix में। Diagonal = $a_{11}, a_{22}, \ldots$
शून्य आव्यूह (Zero — O) $a_{ij} = 0$ सब के लिए सभी elements = 0। "O" से लिखते हैं, "0" से नहीं! $A + O = A,\ AO = O$
विकर्ण आव्यूह (Diagonal) $a_{ij}=0$ जब $i\neq j$ Diagonal के बाहर सभी elements = 0 Square matrix होना ज़रूरी
अदिश आव्यूह (Scalar) Diagonal + $a_{ii}$ = k (equal) Diagonal के सभी elements समान। Diagonal matrix का special case। Identity ⊂ Scalar ⊂ Diagonal
तत्समक आव्यूह (Identity — I) $a_{ii}=1,\ a_{ij}=0\ (i\neq j)$ Diagonal = 1, बाकी = 0। $AI = IA = A$ — कुछ नहीं बदलता। $I_2, I_3$ याद करो directly
ऊपरी त्रिभुजीय (Upper Triangular) $a_{ij}=0$ जब $i > j$ Diagonal के नीचे सभी elements = 0 —
निचली त्रिभुजीय (Lower Triangular) $a_{ij}=0$ जब $i < j$ Diagonal के ऊपर सभी elements = 0 —

Formula का नाम Formula / Condition एक-line meaning Note
समानता शर्त (Equality) $A = B$ तभी जब
  • ① Order same: $A$ और $B$ का order बराबर हो
  • ② $a_{ij} = b_{ij}$ सब (i,j) के लिए
 Order अलग हो तो directly Not Equal — elements मत देखो
Topic 4

जोड़, घटाव और अदिश गुणन (Addition, Subtraction & Scalar)

Formula का नाम Formula एक-line meaning Condition
आव्यूह योग (Addition) $(A+B)_{ij} = a_{ij}+b_{ij}$ Same order की matrices के corresponding elements जोड़ो यह तभी होगा जब A और B का order same हो
आव्यूह अंतर (Subtraction) $(A-B)_{ij} = a_{ij}-b_{ij}$ $A - B = A + (-1)B$ — corresponding elements घटाओ Same order ज़रूरी
ऋणात्मक आव्यूह (Negative) $-A = (-1)A$ हर element का sign बदल जाता है $A + (-A) = O$
अदिश गुणन (Scalar Mult.) $(kA)_{ij} = k \cdot a_{ij}$ हर एक element को scalar k से गुणा करो — एक भी नहीं छोड़ना $k$ कोई भी real number हो सकता है
क्रमविनिमेय (Commutative) $A + B = B + A$ जोड़ में order नहीं बदलता Same order ज़रूरी
साहचर्य (Associative) $(A+B)+C = A+(B+C)$ किसी भी order में जोड़ो — result same —
योज्य तत्समक (Additive Identity) $A + O = O + A = A$ Zero matrix जोड़ने से कुछ नहीं बदलता O का order A जैसा होना चाहिए
योज्य प्रतिलोम (Additive Inverse) $A + (-A) = O$ A में −A जोड़ो तो Zero matrix मिलता है $-A = (-1)A$
अदिश वितरण 1 $k(A+B) = kA + kB$ Scalar को दोनों matrices पर distribute कर सकते हैं —
अदिश वितरण 2 $(k+l)A = kA + lA$ दो scalars का sum, अलग-अलग multiply के बराबर है —
अदिश साहचर्य $k(lA) = (kl)A$ दो scalars को पहले multiply करके matrix पर लगाओ —
Topic 5

आव्यूह गुणन (Matrix Multiplication)

Formula का नाम Formula एक-line meaning Condition / Warning
गुणन शर्त (Condition) $A(m\times n) \times B(n\times p)$ AB possible तभी जब A के columns = B के rows (बीच के दोनों n बराबर हों) Result का order = $m \times p$
गुणन सूत्र (Formula) $(AB)_{ij} = \displaystyle\sum_{k=1}^{n} a_{ik} \cdot b_{kj}$ i वीं row के elements × j वें column के elements → सब जोड़ो —
क्रमविनिमेय नहीं (Non-Commutative) $AB \neq BA$ Matrix multiplication में order बदलने से answer बदल जाता है ⚠️ Generally AB ≠ BA
साहचर्य (Associative) $(AB)C = A(BC)$ तीन matrices का गुणन किसी भी order में — result same —
वितरण नियम (Distributive) $A(B+C) = AB+AC$ A को B और C में distribute कर सकते हैं $(B+C)A = BA+CA$ भी
तत्समक गुण (Identity) $AI = IA = A$ Identity matrix से गुणा करने पर matrix नहीं बदलती I का order match करना चाहिए
शून्य गुण (Zero Property) $AO = OA = O$ Zero matrix से गुणा करने पर Zero ही मिलता है —
शून्य गुणनफल जाल (Zero Product Trap) $AB = O \not\Rightarrow A=O$ या $B=O$ $AB = O$ मिले तो इसका मतलब A या B का Zero होना ज़रूरी नहीं! ⚠️ Numbers जैसा rule यहाँ नहीं चलता
$A^2$ का अर्थ $A^2 = A \times A$ Matrix को खुद से एक बार multiply करो — elements को individually square नहीं करते! ⚠️ Bihar Board Trap!
अदिश गुणन नियम $(kA)B = k(AB) = A(kB)$ Scalar को गुणन से पहले या बाद में कहीं भी रख सकते हैं —
Topic 6

परिवर्त (Transpose)

Formula का नाम Formula एक-line meaning Note
परिवर्त (Transpose) $A' = [a_{ji}]_{n\times m}$ Rows → Columns, Columns → Rows बन जाती हैं। Order $m\times n \to n\times m$ हो जाता है। $A'$ या $A^T$ दोनों notation चलते हैं
द्विगुण परिवर्त (Double Transpose) $(A')' = A$ दो बार transpose करने पर original matrix वापस मिलती है —
अदिश परिवर्त (Scalar Transpose) $(kA)' = kA'$ Scalar को transpose के बाहर रख सकते हैं —
योग परिवर्त (Addition Transpose) $(A+B)' = A'+B'$ जोड़ का transpose = अलग-अलग transpose का जोड़ Order same होना ज़रूरी
गुणन परिवर्त (Product Transpose) $(AB)' = B'A'$ गुणन का transpose निकालते हैं तो order REVERSE होती है — पहले B', फिर A' ⚠️ $(AB)' \neq A'B'$
तीन matrices का परिवर्त $(ABC)' = C'B'A'$ तीन का transpose — सब उल्टा क्रम में —
Topic 7

सममित और विषम सममित आव्यूह (Symmetric & Skew Symmetric)

Formula का नाम Formula / Condition एक-line meaning Important Note
सममित आव्यूह (Symmetric) $A' = A$ Transpose करने पर matrix नहीं बदलती — यानी $a_{ij} = a_{ji}$ सब के लिए Square matrix होना ज़रूरी। Diagonal पर कोई भी value।
विषम सममित (Skew Symmetric) $A' = -A$ Transpose = negative। यानी $a_{ij} = -a_{ji}$ सब के लिए ⭐ Diagonal के सभी elements = 0 (हमेशा!)
विकर्ण का प्रमाण (Diagonal Proof) $a_{ii} = -a_{ii} \Rightarrow a_{ii} = 0$ Skew में diagonal 0 क्यों होता है — यही proof है —
$(A+A')$ का स्वभाव $(A+A')' = A+A'$ $(A+A')$ हमेशा Symmetric matrix होती है —
$(A-A')$ का स्वभाव $(A-A')' = -(A-A')$ $(A-A')$ हमेशा Skew Symmetric matrix होती है —
योगफल विभाजन (Sum Decomposition) $A = \dfrac{1}{2}(A+A') + \dfrac{1}{2}(A-A')$ हर square matrix को Symmetric + Skew Symmetric के योग में लिखा जा सकता है ⚠️ 4-5 marks का important theorem!
Quick Check: किसी matrix को देखते ही Skew check करना हो → पहले diagonal देखो। Diagonal में कोई भी non-zero element है? → Skew Symmetric हो ही नहीं सकता।
Topic 8

प्रारंभिक संक्रियाएँ (Elementary Operations)

प्रकार Notation Meaning Example
① अदला-बदली (Interchange) $R_i \leftrightarrow R_j$
$C_i \leftrightarrow C_j$		 दो rows (या columns) को आपस में बदल दो $R_1 \leftrightarrow R_2$ — पहली और दूसरी row swap
② अदिश गुणन (Scale) $R_i \to kR_i$
$C_i \to kC_i$		 Row/Column के सभी elements को non-zero k से गुणा करो $k \neq 0$ ज़रूरी
③ योग (Add) $R_i \to R_i + kR_j$
$C_i \to C_i + kC_j$		 एक row में दूसरी row का k गुना जोड़ो $R_2 \to R_2 - 2R_1$
SSA याद करो: Swap → Scale → Add। बस तीन types — Row पर भी, Column पर भी।
Topic 9

व्युत्क्रम आव्यूह (Invertible Matrix — Inverse)

Formula का नाम Formula एक-line meaning Condition
व्युत्क्रम की परिभाषा (Definition) $AB = BA = I \Rightarrow B = A^{-1}$ A और B का गुणनफल Identity हो तो B, A का inverse है A square matrix होनी चाहिए
अद्वितीयता (Uniqueness) Inverse is unique एक matrix का inverse हमेशा एक ही होता है — दो नहीं हो सकते —
गुणन व्युत्क्रम (Product Inverse) $(AB)^{-1} = B^{-1}A^{-1}$ गुणन का inverse निकालते हैं तो order REVERSE होती है — पहले $B^{-1}$, फिर $A^{-1}$ ⚠️ $(AB)^{-1} \neq A^{-1}B^{-1}$
परिवर्त-व्युत्क्रम संबंध $(A')^{-1} = (A^{-1})'$ Transpose का inverse = Inverse का transpose — दोनों equal हैं —
Identity का व्युत्क्रम $I^{-1} = I$ Identity matrix का inverse वह खुद होती है —
पंक्ति-रूपांतरण विधि (Row Method) $A = IA$ → Row ops → $I = A^{-1}A$ $A = IA$ लिखो → Row operations से Left को I बनाओ → Right side = $A^{-1}$ सिर्फ Row operations — Column mix नहीं
स्तम्भ-रूपांतरण विधि (Column Method) $A = AI$ → Column ops → $I = AA^{-1}$ $A = AI$ लिखो → Column operations से Left को I बनाओ → Right side = $A^{-1}$ सिर्फ Column operations — Row mix नहीं
⚠️ Inverse कब exist नहीं करता: Operations करते वक्त किसी row (या column) के सभी elements 0 हो जाएँ → "Inverse does not exist" लिखो और रुको। | Inverse सिर्फ Square Matrix का होता है।
Topic 10 — Quick Reference

Addition vs Multiplication — Property Comparison

Property Addition (जोड़) Multiplication (गुणन)
Commutative $A+B = B+A$ ✓ $AB \neq BA$ ✗
Associative $(A+B)+C = A+(B+C)$ ✓ $(AB)C = A(BC)$ ✓
Distributive — $A(B+C) = AB+AC$ ✓
Identity $A + O = A$ $AI = IA = A$
Inverse $A + (-A) = O$ $AB = I \Rightarrow B = A^{-1}$
Transpose Rule $(A+B)' = A'+B'$ $(AB)' = B'A'$ ← Reverse!
Inverse Rule — $(AB)^{-1} = B^{-1}A^{-1}$ ← Reverse!
