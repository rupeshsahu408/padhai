Ex 1.1

संबंधों के प्रकार (Types of Relations)

Reflexive Symmetric Transitive Equivalence Empty/Universal
1
Reflexive
$\mathbb{R}$ में संबंध $R = \{(a,b) : a \leq b\}$ के बारे में कौन सा कथन सत्य है?
A
Reflexive और Symmetric दोनों
B
Reflexive और Transitive, पर Symmetric नहीं
C
केवल Symmetric
D
Equivalence Relation
✓ सही Answer: B
Explanation
Reflexive: $a \leq a$ हमेशा सत्य है → Reflexive ✓
Symmetric नहीं: $1 \leq 3$ है, लेकिन $3 \leq 1$ नहीं → Symmetric ✗
Transitive: $a \leq b$ और $b \leq c$ हो तो $a \leq c$ → Transitive ✓
Equivalence नहीं क्योंकि Symmetric fail हुआ।
2
Symmetric
$\mathbb{N}$ में $R = \{(a,b) : a \text{ भाजक है } b \text{ का}\}$। यह संबंध है:
A
Reflexive और Symmetric
B
Reflexive और Transitive पर Symmetric नहीं
C
Symmetric और Transitive
D
Equivalence Relation
✓ सही Answer: B
Explanation
Reflexive: हर $a$, $a$ का भाजक है → ✓
Symmetric नहीं: $2 \mid 4$ है ($2$ भाजक है $4$ का), लेकिन $4 \mid 2$ नहीं → ✗
Transitive: $a \mid b$ और $b \mid c$ हो तो $a \mid c$ → ✓
3
Equivalence Relation
$\mathbb{Z}$ में $R = \{(a,b) : 5 \mid (a-b)\}$। यह संबंध है:
A
केवल Reflexive
B
Reflexive और Symmetric, Transitive नहीं
C
Equivalence Relation
D
इनमें से कोई नहीं
✓ सही Answer: C
Explanation
Reflexive: $5 \mid (a-a) = 0$ → ✓
Symmetric: $5 \mid (a-b)$ तो $5 \mid (b-a)$ → ✓
Transitive: $5 \mid (a-b)$ और $5 \mid (b-c)$ तो $5 \mid (a-c)$ → ✓
तीनों satisfied → Equivalence Relation ✓
4
Empty / Universal Relation
$A = \{1,2,3\}$ पर रिक्त संबंध ($R = \phi$) के बारे में कौन सत्य है?
A
Reflexive है
B
Symmetric और Transitive दोनों हैं, Reflexive नहीं
C
Equivalence Relation है
D
तीनों (R, S, T) fail करता है
✓ सही Answer: B
Explanation
रिक्त संबंध में कोई pair ही नहीं है।
Reflexive नहीं: $(1,1) \notin \phi$ → ✗
Symmetric: कोई $(a,b) \in \phi$ ही नहीं तो violate भी नहीं होगा → Vacuously True ✓
Transitive: Same reason → Vacuously True ✓
Note: सार्वत्रिक संबंध Equivalence होता है, रिक्त नहीं।
5
Equivalence Class
$\mathbb{Z}$ में $R = \{(a,b) : 2 \mid (a-b)\}$ में तुल्यता वर्ग $[1]$ होगा:
A
$\{0, 2, 4, 6, ...\}$
B
$\{1, 3, 5, 7, ...\}$
C
सभी विषम पूर्णांक
D
B और C दोनों सही हैं
✓ सही Answer: D
Explanation
$[1] = \{a \in \mathbb{Z} : (a,1) \in R\} = \{a : 2 \mid (a-1)\}$
यानी $a-1$ सम हो → $a$ विषम हो।
$[1] = \{..., -3, -1, 1, 3, 5, 7, ...\}$ = सभी विषम पूर्णांक।
B और C दोनों same set को अलग तरह describe कर रहे हैं → D सही।
Ex 1.2

फलनों के प्रकार (Types of Functions)

One-One Onto Bijective Into Finite Set
1
One-One
$f : \mathbb{R} \to \mathbb{R}$, $f(x) = x^2$ के बारे में सत्य है:
A
एकैकी है
B
आच्छादक है
C
एकैकी नहीं है
D
एकैकी आच्छादी है
✓ सही Answer: C
Explanation
$f(-2) = 4$ और $f(2) = 4$, लेकिन $-2 \neq 2$।
दो अलग inputs का एक ही output → Many-One है, एकैकी नहीं।
Also: Range $= [0,\infty) \neq \mathbb{R}$ → Onto भी नहीं।
2
Onto (Surjective)
$f : \mathbb{R} \to \mathbb{R}$, $f(x) = 2x + 5$ है। यह फलन है:
A
केवल एकैकी
B
केवल आच्छादक
C
एकैकी आच्छादी (Bijective)
D
Neither One-One nor Onto
✓ सही Answer: C
Explanation
One-One: $f(x_1)=f(x_2) \Rightarrow 2x_1+5=2x_2+5 \Rightarrow x_1=x_2$ ✓
Onto: कोई $y \in \mathbb{R}$ लो, $x = \frac{y-5}{2} \in \mathbb{R}$ exist करता है ✓
दोनों satisfied → Bijective।
3
Into Function
कौन सा फलन Into (आच्छादक नहीं) है?
A
$f:\mathbb{R}\to\mathbb{R}$, $f(x)=2x+1$
B
$f:\mathbb{R}^+\to\mathbb{R}^+$, $f(x)=x^2$
C
$f:\mathbb{R}\to\mathbb{R}$, $f(x)=x^2$
D
$f:\mathbb{R}\to\mathbb{R}$, $f(x)=x^3$
✓ सही Answer: C
Explanation
$f(x)=x^2$ का Range $= [0,\infty)$ जो $\mathbb{R}$ के बराबर नहीं।
$y = -4$ के लिए $x = \sqrt{-4} \notin \mathbb{R}$ → Onto नहीं → Into है।
A, D: Bijective हैं। B: $\mathbb{R}^+$ पर $x^2$ Bijective है।
4
Finite Set — One-One ↔ Onto
$n(A) = n(B) = 3$ और $f : A \to B$ एकैकी है, तो $f$ जरूर होगा:
A
केवल एकैकी, Onto नहीं
B
एकैकी आच्छादी (Bijective)
C
Onto पर एकैकी नहीं
D
कुछ नहीं कहा जा सकता
✓ सही Answer: B
Explanation
Finite sets में जब $n(A) = n(B)$ हो:
One-One $\iff$ Onto $\iff$ Bijective
$A$ के 3 elements, $B$ के 3 elements — अगर One-One है तो तीनों elements अलग-अलग map होंगे → automatically सभी $B$ के elements cover → Onto ✓
Ex 1.3

संयोजन और प्रतिलोम (Composition & Inverse)

gof / fog Invertible f⁻¹ (gof)⁻¹ Composition Property
1
gof value
$f(x) = 2x+1$ और $g(x) = x^2-1$ हो, तो $(g \circ f)(2)$ का मान होगा:
A
$24$
B
$8$
C
$5$
D
$3$
✓ सही Answer: A
Explanation
Step 1: $f(2) = 2(2)+1 = 5$
Step 2: $g(f(2)) = g(5) = 5^2 - 1 = 25-1 = 24$
Key: $g \circ f$ में पहले $f$ apply करो, फिर $g$।
2
Invertible condition
$f : A \to B$ व्युत्क्रमणीय (Invertible) होगा यदि और केवल यदि:
A
$f$ एकैकी हो
B
$f$ आच्छादक हो
C
$f$ एकैकी आच्छादी हो
D
$f$ Onto न हो
✓ सही Answer: C
Explanation
$f^{-1}$ exist करने के लिए दोनों शर्तें जरूरी हैं:
• One-One: वरना $f^{-1}(c)$ के लिए दो values होंगी — ambiguity।
• Onto: वरना कुछ $b \in B$ के लिए $f^{-1}(b)$ undefined रहेगा।
इसलिए Bijective होना जरूरी है।
3
(gof)⁻¹
यदि $f$ और $g$ दोनों Bijective हों, तो $(g \circ f)^{-1}$ बराबर है:
A
$g^{-1} \circ f^{-1}$
B
$f^{-1} \circ g^{-1}$
C
$f \circ g$
D
$g \circ f$
✓ सही Answer: B
Explanation
$(g \circ f)^{-1} = f^{-1} \circ g^{-1}$ — order उलटा होता है।
याद करो: मोज़े पहले, जूता बाद में — निकालते वक्त जूता पहले, मोज़ा बाद में।
$g$ बाद में लगा था → $g^{-1}$ पहले लगेगा — लेकिन यहाँ notation में $f^{-1}$ पहले लिखते हैं।
4
gof One-One → f One-One
यदि $g \circ f$ एकैकी है, तो निश्चित रूप से:
A
$g$ एकैकी है
B
$f$ एकैकी है
C
$f$ और $g$ दोनों एकैकी हैं
D
$f$ आच्छादक है
✓ सही Answer: B
Explanation
$g \circ f$ One-One है → $f$ One-One है (theorem)।
लेकिन $g$ के बारे में कुछ नहीं कह सकते — $g$ One-One भी हो सकता है, नहीं भी।
Counter-example: $f:\{1,2\}\to\{1,2,3\}$ One-One, $g:\{1,2,3\}\to\{1,2\}$ Many-One, फिर भी $gof$ One-One हो सकता है।
Ex 1.4

द्विआधारी संक्रियाएँ (Binary Operations)

Commutative Associative Identity Inverse
1
Commutative
$\mathbb{Z}$ में $a * b = a - b$ क्रमविनिमेय है या नहीं?
A
हाँ, क्रमविनिमेय है
B
नहीं, क्रमविनिमेय नहीं है
C
कुछ elements के लिए है
D
केवल $a = b$ होने पर
✓ सही Answer: B
Explanation
$a * b = a - b$ लेकिन $b * a = b - a$।
Example: $3 * 5 = 3-5 = -2$, लेकिन $5 * 3 = 5-3 = 2$।
$-2 \neq 2$ → क्रमविनिमेय नहीं है।
2
Identity Element
$\mathbb{Q}$ में $a * b = ab + 1$ का तत्समक अवयव है:
A
$0$
B
$1$
C
$-1$
D
तत्समक अवयव exist नहीं करता
✓ सही Answer: D
Explanation
Identity के लिए: $a * e = a \Rightarrow ae + 1 = a \Rightarrow ae = a-1 \Rightarrow e = \dfrac{a-1}{a}$
$e$ का value $a$ पर depend करता है — fixed नहीं है।
इसलिए कोई एक fixed identity element exist नहीं करता।
3
Inverse Element
$\mathbb{Q} - \{-1\}$ में $a * b = a + b + ab$ है। तत्समक अवयव $e = 0$ है। $a$ का प्रतिलोम होगा:
A
$-a$
B
$\dfrac{1}{a}$
C
$\dfrac{-a}{1+a}$
D
$\dfrac{a}{1-a}$
✓ सही Answer: C
Explanation
$a * a' = 0$ (identity)
$a + a' + aa' = 0$
$a'(1 + a) = -a$
$a' = \dfrac{-a}{1+a}$ ✓
Note: $a \neq -1$ इसलिए $(1+a) \neq 0$ → defined है।
4
Associative
$\mathbb{N}$ में $a * b = 2^{ab}$ साहचर्य है या नहीं?
A
हाँ, साहचर्य है
B
नहीं, साहचर्य नहीं है
C
क्रमविनिमेय है इसलिए साहचर्य भी है
D
कभी-कभी साहचर्य है
✓ सही Answer: B
Explanation
$(a*b)*c = 2^{ab} * c = 2^{2^{ab} \cdot c}$
$a*(b*c) = a * 2^{bc} = 2^{a \cdot 2^{bc}}$
Example: $a=1,b=1,c=2$:
$(1*1)*2 = 2^1*2 = 2^{2} = 4$ → LHS $= 2^{4}=16$
$1*(1*2) = 1*2^2 = 1*4$ → RHS $= 2^{4}=16$
एक example से नहीं, general proof करो — generally equal नहीं → B
